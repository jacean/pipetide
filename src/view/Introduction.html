Introduction

About this bioinformatics prediction platform:

The bioinformatics prediction platform is established for providing useful tools based on computational methods for following bioinformatics researches:

  1. Identification of antimicrobial peptides (AMPs) and their function types
  2. Prediction of protein subcellular localization
  3. Prediction of protein post-translational modification sites

The web-server can complete above tasks online and provides free predicting services for public. Especially, the platform contains a database of antimicrobial peptides as a tool for AMPs study.

NOTE: The FASTA format for the current predictor can be described as follows.

  1. The first line of each query protein input format must begin with a greater-than (">") symbol in the first column. The word following the ">" symbol is the identifier and description of the sequence, but both are optional.
  2. The sequence (in single-character code) begins in a different line and ends if another line starting with a ">" appears, which indicates the start of another query protein
